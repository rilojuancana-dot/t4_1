# t4_1

## Descripción

Esta aplicación ayuda a gestionar los pedidos de un restaurante, guardando la información de los mismos y calculando el total. 

## Tecnologías

El proyecto está hecho enteramente usando el framework de **Flutter**. 

## Funcionalidad

Su utilidad es más de demostración, ya que es un proyecto didáctico. Por ejemplo, los valores de `Producto` están hardcodeados, la UI y UX no están casi trabajadas, hay funcionalidades sin implementar enteramente, etc.
